# イヤポイ (Iyapoi) 🗑️✨

嫌な気持ちを言語化し、物理的に「捨てる」アクションを通じて心を整えるセルフケアアプリのモックアップです。

## 💡 コンセプト
「心の汚れを溜め込まない」
モヤモヤした感情を書き出し、ゴミ箱に捨てるという「儀式」を行うことで、心理的な切り離しを促します。

## 🚀 機能
- **感情の数値化:** 今の不快度をスコアで入力。
- **ゴミ出しアニメーション:** 書いたテキストが消える演出。
- **リフレーム:** 捨てた後に優しいメッセージを表示。

## 🛠️ 使用技術
- HTML5 / CSS3 (Tailwind CSS)
- JavaScript
